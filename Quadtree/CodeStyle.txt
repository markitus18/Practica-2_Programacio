FileName:
- Will use PascalCase
- Example: Main.cpp

Variables:
- Will use camelCase
- Example: movementSpeed

Class Names:
- Will use PascalCase
- Example: Point

Method Names:
- Will use PamelCase
- Example: IsZero()

	before the method name with an underscore between the identifier and the method name
	- Example: Node_PreorderRecursive
